# 🌍 Travel Website (Mayar Travel)

A modern travel website built using **HTML, CSS, and Bootstrap**.  
It showcases travel packages, destinations, services, and a booking form.

## 🚀 Features
- Responsive navigation bar with scroll spy
- Hero section with animated carousel
- About section describing the service
- Booking form for trip reservations
- Travel packages with pricing
- Services section (food, hotel, travel)
- Image gallery of destinations
- Footer with social media links
- Fully responsive design using Bootstrap

## 🛠️ Technologies Used
- HTML5
- CSS3
- Bootstrap 5
- Bootstrap Icons
- Font Awesome
